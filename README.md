# shell-utils

shell-utils is a toolkit that I use in my daily routine. I created this
repository so that I can setup my new environment quickly whenever I need.

## Install

### Install with cURL

```sh
curl -o- https://raw.githubusercontent.com/teyfik-on-it/shell-utils/17858424bc0019c5f839e925bc9377e11a3f002b/install.sh | bash
```

### Install with wget

```sh
wget -qO- https://raw.githubusercontent.com/teyfik-on-it/shell-utils/17858424bc0019c5f839e925bc9377e11a3f002b/install.sh | bash
```
